Parallelization of Irregular Algorithms in GPU Architectures

Bachelor’s Thesis – Computer Science
Polytechnic University of Madrid

Overview:

This project explores parallelization strategies for irregular algorithms on GPU architectures using CUDA, focusing on the computational challenges introduced by non-uniform data structures and workload distribution.
The work implements and evaluates different approaches to solving the N-body problem, comparing CPU and GPU performance for both regular and irregular algorithms. The goal is to analyse how GPU parallelisation techniques can improve computational efficiency and scalability when handling complex simulations.

Two algorithmic approaches were implemented:
-Regular Algorithm – direct all-pairs computation
-Irregular Algorithm – quadratic tree (Barnes–Hut inspired) spatial partitioning

Their performance was evaluated across CPU and GPU implementations to analyse speedup, efficiency, and scalability.
